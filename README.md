# Customer-Behaviour-Analysis
Customer Shopping Behavior Analysis using Python, SQL, and Tableau to explore purchasing patterns and generate business insights. Includes EDA, advanced SQL for segmentation and revenue analysis, and an interactive Tableau dashboard to visualize trends, customers, and product performance.

Dashboard: https://public.tableau.com/app/profile/vedant.sharma7827/viz/CustomerBehaviorDashboard_17702693937200/Dashboard1
Data Analytics Project – Customer Shopping Behavior Analysis
Overview
This project analyzes customer shopping behavior using Python, SQL, and Tableau. It covers exploratory data analysis (EDA), data cleaning, advanced SQL queries, and interactive dashboard creation to identify purchasing patterns, customer segments, and business insights.
Dataset
The dataset contains customer demographics and shopping details such as age, gender, product category, purchase amount, subscription status, shipping type, discounts, and review ratings. It is provided in CSV format and used across Python, SQL, and Tableau for analysis and visualization.
Tools
Python (Pandas, Jupyter Notebook)
SQL (Azure Data Studio / SQL Server)
Tableau
Steps
Loading the Dataset – Imported CSV data into Python using Pandas.
Exploratory Data Analysis (EDA) – Analyzed distributions, correlations, and customer behavior patterns.
Data Cleaning – Handled missing values, duplicates, and incorrect data types.
SQL Analysis – Performed segmentation, revenue analysis, and advanced queries using SQL.
Data Visualization – Built an interactive Tableau dashboard to present insights.
Dashboard
The Tableau dashboard includes KPI cards, revenue analysis, product trends, subscription behavior, and customer segmentation with interactive filters.
Results
Identified high-spending customer segments
Analyzed subscriber vs non-subscriber behavior
Found top-performing products and categories
Observed relationships between purchase frequency and spending
How to Run
Clone the repository.
Install required Python libraries.
Run the Jupyter Notebook for analysis.
Execute SQL queries in Azure Data Studio.
Open Tableau dashboard to explore visual insights.
